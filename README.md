# Agri Market System

## Project Overview

Agri Market System is a web-based platform that connects farmers and customers directly for buying and selling agricultural products.

The system allows farmers to upload products, customers to browse products, add items to cart, place orders, and track order status.

## Features

### Administrator

* Manage users
* Manage products
* Manage orders
* View reports and statistics

### Farmer

* Add products
* Manage products
* View customer orders

### Customer

* Browse products
* Add products to cart
* Checkout and place orders
* Track orders

## Technologies Used

* PHP
* MySQL / MariaDB
* HTML
* CSS
* XAMPP

## Database Setup

1. Create database:

CREATE DATABASE agri_market;

2. Import:

database/agri_market.sql

## Installation

1. Copy project to htdocs.
2. Start Apache and MySQL.
3. Import the database.
4. Configure db.php.
5. Open the application in a browser.
